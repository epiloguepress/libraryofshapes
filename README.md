##Flatland##

###Requirements###

-   git
-   bower
-   npm

###Getting started###

`git clone https://github.com/arodic/flatland.git`

`cd flatland`

`bower install`

`python -m SimpleHTTPServer`

###Directory structure###

**app/** Application code.

**prototypes/** Various prototypes.

**components/** third-party dependencies installed via bower.

**lib/** javascript libraries.

**text/** Flatland text.
